# Workstation Setup

## Entry Checks
- macOS or Windows with admin rights
- Git and GitHub account
- Docker Desktop available

## Steps
1) Install Docker Desktop. Enable Kubernetes only if needed later.  
2) Install Git and GitHub CLI:
   ```bash
   brew install git gh || choco install git gh
   gh auth login
   ```
3) Python and MkDocs:
   ```bash
   pyenv install -s 3.11.9 || true
   python3 -m venv .venv && source .venv/bin/activate
   pip install -U pip mkdocs mkdocs-material
   ```
4) Validate:
   ```bash
   mkdocs --version
   docker version
   gh --version
   ```

## Exit Criteria
- `mkdocs serve` works
- Docker pulls and runs hello-world
- `gh auth status` shows logged in
